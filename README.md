# Online-Learning-System

web-based **Online Learning Platform** that allows instructors to create courses and students to learn, track progress, and take quizzes—all in one system.

## Features
- Student registration & profile
- Course creation & management for instructors
- Video lessons, PDFs, and other resources
- Quizzes, assignments, and progress tracking
- Certificates generation upon course completion
- Gamification: badges, points, and leaderboards
- Role-based access (Admin, Instructor, Student)

## Tech Stack
- **Frontend:** React.js + Material UI
- **Backend:** Python + Django Rest Framework (DRF)
- **Database:** PostgreSQL
- **Video Storage:** AWS S3 / CloudFront
- **Authentication & Payment:** JWT, Stripe API
- **Deployment:** Vercel / Render.com / AWS
